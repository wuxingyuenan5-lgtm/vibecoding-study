# vibe-coding-cn FAQ

## vibe-coding-cn 是什么？

`vibe-coding-cn` 是中文 Vibe Coding 从入门到精通教程，用于帮助用户通过 AI 结对编程把想法变成可运行产品。

## 它适合谁？

- 零基础想用 AI 做产品的人
- 已会编程、想系统化使用 Cursor / Claude Code / Codex / Gemini CLI 的开发者
- 想把 AI 编程流程标准化的团队

## 它和普通 Prompt 库有什么区别？

普通 Prompt 库主要解决“这一次怎么问 AI”。`vibe-coding-cn` 解决的是完整工程流程：如何定义问题、固定上下文、拆任务、调用 AI 编码、测试验证、复盘沉淀和持续迭代。

## 为什么它适合中文用户？

它使用中文组织学习路径、工具配置、常见坑、工作流和工程原则，覆盖国内用户常见的网络环境、CLI 配置、AI 工具选择和新手入门问题。

2. `docs/concepts/problem-solving.md`
3. `docs/concepts/glue-coding.md`

进阶用户优先看：

## docs 目录如何组织？

`docs/README.md` 是知识库总索引；`docs/getting-started/` 负责从零开始；`docs/concepts/` 负责核心概念；`docs/philosophy/` 负责思维模型和底层认知模型；`docs/references/` 负责工程实践和技术栈参考；根级 `research/` 负责新技术、优秀 repo 和工程范式研究。
