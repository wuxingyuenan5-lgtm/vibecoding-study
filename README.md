# Vibe Coding 学习区

这是代码新手的 AI 编程练习目录。

核心目标：

```text
用 AI 把想法稳定做成可运行、可修改、可复盘的 Web 原型。
```

## 学习者画像

你不是单纯从零找方向的新手。你的真实背景是：

- 代码基础薄弱。
- 正在初创一个私募基金平台业务。
- 业务需求相对明确，知道自己想做什么。
- 短板主要在工程化表达、专业技术名词、项目管理、质量验收、代码协作边界。

因此，本学习区的重点不是泛泛学编程，而是提高 Vibe Coding 的专业水平：把业务需求翻译成工程任务，把 AI 输出纳入可控流程，并逐步补齐项目管理和技术沟通能力。

## 当前主线

第一阶段只做一件事：

```text
跑通一个小游戏 -> 改小功能 -> 修 bug -> 写学习日志
```

小游戏只是训练闭环，不是最终目标。

第二阶段主项目：

```text
私募基金平台相关的低风险练习模块
```

优先从不涉及真实资金、真实账户、真实客户隐私的模块练起，例如交易复盘、研究笔记、产品需求看板、运营后台原型。不要同时开多个项目。一个项目连续迭代，比三个半成品更有训练价值。
