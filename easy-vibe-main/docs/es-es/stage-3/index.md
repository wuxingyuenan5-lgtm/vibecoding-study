## Para quién es

- Desarrolladores avanzados con experiencia en desarrollo full-stack que quieren desafiar aplicaciones más complejas
- Ingenieros que quieren dominar tecnologías de desarrollo multiplataforma
- Exploradores que quieren comprender profundamente el desarrollo de aplicaciones nativas de IA
- Blogueros técnicos que quieren establecer su marca técnica personal

## Requisitos previos

- Completar la etapa de "Desarrollo Junior a Intermedio", o tener experiencia en desarrollo full-stack
- Estar familiarizado con frameworks frontend (como React/Vue) y desarrollo backend
- Comprender conceptos básicos de IA y el uso de APIs
