### 启动与基础显示

- [ ] 页面可以正常打开，无明显报错。

### 游戏控制

- [ ] 点击开始后，蛇按固定速度移动。
- [ ] 四个方向键均能改变移动方向。
- [ ] 不能直接从当前方向反向移动并撞入自身。
- [ ] 重复点击开始不会产生多个定时循环。

### 食物与分数

- [ ] 蛇吃到食物后身体增加一节。
- [ ] 吃到食物后分数按规则增加。
- [ ] 新食物不会生成在蛇身上。

- [ ] 撞墙后立即结束游戏。
- [ ] 撞到自身后立即结束游戏。
- [ ] 游戏结束后不再继续移动或加分。
- [ ] 点击重新开始后，蛇、方向、食物、分数和游戏状态均被重置。
