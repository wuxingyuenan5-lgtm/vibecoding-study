# 练习区说明

本目录用于保存正式课程、作业、实践代码、验收记录和复盘文档。

## 目录结构

```text
练习文件夹/
├─ 00-学习管理/
│  ├─ 课程总览.md
│  ├─ 学习进度.md
│  └─ 通用术语表.md
└─ 01-snake-game-01/
   ├─ 01-教学讲义.md
   ├─ 02-作业要求.md
   ├─ 03-实施计划.md
   ├─ 04-验收清单.md
   ├─ 05-学习日志.md
   ├─ 06-老师评语.md
   └─ README.md
```

## 固定学习闭环

1. 阅读教学讲义和作业要求。
2. 先写实施计划，不立即大范围生成代码。
3. 明确修改文件、风险和验收方法。
4. 在本地完成开发与测试。
5. 更新验收清单和学习日志。
6. 提交并推送 GitHub。
7. 老师读取仓库并将审查意见写入老师评语。
8. 根据评语修正，再次提交。

## 文件边界

- 三个资料库只用于阅读和查找资料，不直接写入练习代码。
- 每个课程项目都使用独立文件夹。
- 真实账户、客户隐私、API Key、生产数据不得提交。
- 第一阶段只使用模拟数据和本地运行环境。
